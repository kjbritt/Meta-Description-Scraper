# Meta-Description-Scraper
A Python script to scrape meta descriptions from a TXT file of URLs and output them to a CSV

## Using the Script

You will need Python installed on your machine in order to run this script. From your machine's command line, type meta_description_scraper.py and hit enter. The script will run and prompt you for the name of a TXT file. The TXT file should contain only a list of the URLs you wish to scrape, with each URL on new line. Type the name of the TXT file, hit enter, and the script will produce a CSV file named meta_descriptions.csv.
